Software alternatives

i decided to leave some of the hnhx's alternatives since i find them useful.

Operating systems

Avoid Windows and macOS, instead use any GNU/Linux distribution or *BSD variant. If you want a 100% free system check out the FSF endorsed distributions.

VPN providers

Don't use one, instead use Tor or I2P, if thats not an option buy IVPN, OVPN or Mullvad VPN preferably via Monero or cash.

Web browsers

If you like Firefox switch to LibreWolf or use hnhx's user.js for Firefox, if you like Chrome switch to Ungoogled Chromium, if you are on Android use Cromite

Discord and instant messaging services

If you want something modern with multimedia, screenshare, VC and encryption use Matrix with the Element client, if you don't like electron use nheko if you still like Discord use betterdiscord for Windows And betterdiscordctl for Linux with the DoNotTrack plugin, for Android use Aliucord

Botnets

Avoid using apps like Instagram, Facebook, Twitter... Use an open source search engine like LibreY, use XMANAGER for Spotify Android, use VSCodium (spyware free VS Code), also check out NewPipe for a private YouTube for Android.

Clouds

Self host your own cloud

Android

Use GrapheneOS, if you don't like it use LineageOS

instead.

Other

Use Optimizer for debloating Windows and making it (kinda) more private, Universal-Android-Debloater for uninstalling Google spyware (bloatware) and OEM apps
