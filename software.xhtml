Software alternatives

i decided to leave some of the hnhx's alternatives since i find them useful.

Operating systems

Avoid Windows and macOS, instead use any GNU/Linux distribution or *BSD variant. If you want a 100% free system check out the FSF endorsed distributions.

VPN providers

Don't use one, instead use Tor or I2P, if thats not an option buy IVPN, OVPN or Mullvad VPN preferably via Monero or cash.

Web browsers

If you like Firefox switch to LibreWolf or use hnhx's user.js for Firefox, if you like Chrome switch to Ungoogled Chromium, if you are on Android use Cromite

Discord and instant messaging services

If you want something modern with multimedia, screenshare, VC and encryption use Matrix with the Element client, if you don't like electron use nheko if you still like Discord use betterdiscord for Windows And Vencord, for Android use Aliucord.

Botnets

Avoid using apps like Instagram, Facebook, Twitter... Use an open source search engine like LibreY or SearXNG, use XMANAGER for Spotify Android, use VSCodium (spyware free VSCode), also check out NewPipe for a private YouTube for Android.

Clouds

Self host your own cloud.

Android

Use GrapheneOS, if you don't like it use LineageOS , Universal-Android-Debloater for uninstalling Google spyware (bloatware) and OEM apps.

Other

Ventoy for make a bootable USB with multiplate ISOs. Privacy.sexy for make your own debloating and private script depending on what you need, SponsorBlock skip YouTube sponsors, YouTubeAgeBypass bypass YouTube's Age restricted videos.

Windows

If for some reason you still use Windows and like to being spied then i have some things for you.

Tron Script The most complete script for de-bloat , clean and desinfect Windows.

PrivateZilla Simpliest way to make Windows more secure.

Optimizer Simpliest way to debloat Windows in a few clicks.

Microsoft Activation Scripts (MAS) Activator for Windows and Office using HWID.

WizTree See what is taking all your disk space.

ProcessHacker Advanced task manager , good for detecting malware.
