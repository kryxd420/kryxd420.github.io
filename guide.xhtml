This question is actually important, depending on for what do you need Linux im going to recommend you installing it in different ways, for example, if you are going to use just for programming or studying then just a virtual machine or dual-boot, but if you want to learn linux to make it your main OS then delete any other OS and install only Linux, this is because you are going to find a lot of problems, you are going to get frustated and then delete the OS and you will learn nothing.

3. Basic terminal commands.

Sudo

Sudo will be the equivalent of "Admin" in Windows, it allow us to run commands as root, just type "sudo" before some random command and it will execute it with root.

Package managers

Depending on the distro you choosed it will use a different package manager, for arch based distros is "pacman" and for debian and ubuntu ones is "apt"

  • Installing softare:

    • sudo apt-install *package , sudo pacman -S *package

  • Removing software:

    • sudo apt-remove *package , sudo pacman -R *package

  • Updating

    • sudo apt full-upgrade , sudo pacman -Syu/sudo pacman -Syyu

man

man its actually a very helpfull command for begginers, it will display a verbose manual of a command (man *command*)

ls

Lists the content of a directory. (ls /dir)

mv

move files between directorys (mv /dir/file.txt /newdir/file.txt)

cp

copy files between directorus, same as mv but write cp instead of mv.
